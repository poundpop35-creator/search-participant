คู่มือการใช้งานระบบเช็คอิน
1. เลือกวันที่สแกน
  • ระบบจะเลือกวันอัตโนมัติตามวันที่ปัจจุบัน ไม่ต้องปรับเอง
  • วันงาน (24–26 มิ.ย.) — ตัวเลือก "รับกระเป๋า" และ "รับเสื้อ" จะซ่อนอัตโนมัติ เพื่อป้องกันกดผิด
  • ก่อนวันงาน — สามารถเลือกโหมดรับกระเป๋า / รับเสื้อได้
2. วิธีสแกน
  • เครื่องยิงบาร์โค้ด — ยิง QR Code บนบัตรผู้เข้าร่วม หรือพิมพ์เลขบัตรประชาชน / Passport แล้วกดค้นหา
  • กล้องมือถือ — กดปุ่มแล้วส่องกล้องที่ QR Code
3. ผลลัพธ์โหมดเช็คอิน
  • ✅ เช็คอินสำเร็จ — แจ้งสิทธิ์คูปองอาหาร + ของที่ระลึก แจกตามรายการที่ขึ้น
  • ✅ เช็คอินแล้ว โดย admin — ผู้บริหาร / กองแผนงานฯ ได้รับการยกเว้น ไม่ต้องทำอะไร
  • ⚠️ เคยเช็คอินแล้ว — สแกนซ้ำวันเดิม ทุกรายการแสดงสีแดง = แจกไปแล้วทั้งหมด
  • ❌ ไม่พบข้อมูล — ยังไม่มีในระบบ ให้กรอก Walk-in แล้วรอ ~1 นาที
4. ของที่ระลึก (โหมดเช็คอิน)
  • คูปองกระเป๋า (สีส้ม) — มีของที่ระลึกอื่นอยู่ในกระเป๋าด้วย
  • คูปองกระเป๋า (สีเขียว) — ได้กระเป๋าอย่างเดียว
  • ของที่ระลึก (2) จะแสดงชื่อของ เช่น "บทคัดย่อ" หมายถึงอยู่ภายในกระเป๋าแล้ว
  • เสื้อแจ็คเก็ต — แจกพร้อมเช็คอิน ครั้งต่อไปจะขึ้น "รับไปแล้ว" สีแดง
5. โหมดรับกระเป๋า (ก่อนวันงาน)
  • เลือก "รับกระเป๋า" → สแกน → ระบบบันทึกได้ครั้งเดียว
  • ถ้ามีเสื้อที่ยังไม่ได้รับ จะมีปุ่ม 👕 รับเสื้อพร้อมกันด้วย ให้กดทันที
  • ไม่มีสิทธิ์ / รับไปแล้ว — ระบบจะแจ้งและไม่บันทึกซ้ำ
6. โหมดรับเสื้อ (ก่อนวันงาน)
  • เลือก "รับเสื้อ" → สแกน → ระบบบันทึกได้ครั้งเดียว
  • ต้องมีไซต์เสื้อในทะเบียนจึงจะมีสิทธิ์รับ
  • ถ้ารับเสื้อแล้วและมารับกระเป๋า — ระบบรับกระเป๋าจะแสดงสถานะเสื้อว่า "รับไปแล้ว ✓" ให้อัตโนมัติ
7. Walk-in
ผู้เข้าร่วมที่ไม่ได้ลงทะเบียนล่วงหน้า กรอกแบบฟอร์ม Walk-in เอง ข้อมูลจะเข้าระบบอัตโนมัติภายใน ~1 นาที จากนั้น staff สแกนได้เลย
